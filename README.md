# 💰 Smart Expense Tracker

A clean and interactive web application to help users manage their personal finances by tracking daily and monthly income and expenses in real time.

## 🚀 Features

- **Add Income & Expenses** — Log transactions instantly with a description and amount
- **Live Balance Updates** — Account balance updates automatically with every transaction
- **Color-Coded Transactions** — Income appears in green, expenses in red for quick visibility
- **Edit Transactions** — Modify any existing entry without losing your data
- **Delete Transactions** — Remove unwanted entries with a single click; balance adjusts immediately
- **Daily & Monthly Tracking** — Monitor your spending patterns over time

## 🛠️ Tech Stack

- **HTML5** — Structure and layout
- **CSS3** — Styling and color-coded UI
- **JavaScript (Vanilla)** — Logic, DOM manipulation, and real-time updates

## 📸 How It Works

1. Enter a transaction name and amount
2. Submit to instantly see it reflected in your balance
3. Income is highlighted in **green**, expenses in **red**
4. Use the ✏️ edit button to update any transaction
5. Use the ❌ delete button to remove a transaction — balance adjusts automatically

## 🎯 Purpose

Built to practice real-world JavaScript DOM manipulation and develop a functional, user-friendly finance tool.
